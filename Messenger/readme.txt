	Application utilises java sockets to provide chat service via TCP connection.

 	First you should launch Server.jar, which starts listening on port 20000,
 on one of PCs within network (your should tune your firewall to make this port reachable from desired network location).
 Then you can launch Messenger.jar on any PC  from which server PC is reachable, and set server PC's IP in it.
 To do that, you have to invoke dialog by clicking the default IP address label.

	This version allows only public messages and uses numbers in order of connection for automatic naming;
